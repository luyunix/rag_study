## 正文讲解（按视频顺序）

### 1. Agent 概念

LLM 从回答器变成决策器。

### 2. ReAct

推理、行动、观察交替循环。

### 3. 工具集合

多个知识库/检索器作为工具。

### 4. RAG Router

按问题选择正确数据源。

### 5. 实战目标

可控路由、停止与过程日志。


## 用一个例子串起来

用户问制度问题时调用制度知识库，问公司投资关系时调用金融图谱。Agent Router 先判断意图、选择工具、读取结果，再决定是否继续调用或给出答案。
